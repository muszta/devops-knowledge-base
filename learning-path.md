# DevOps Learning Path 🧭

This file defines the recommended learning order for this repository.

---

## 🟢 1. Fundamentals

Start with basics:

- What is DevOps?
- Linux basics
- Networking basics
- Git & GitHub

---

## 🟢 2. Docker

Folder: `docker/`

Learn:
- Containers vs VMs
- Images and Dockerfiles
- Docker Compose
- Best practices

---

## 🟡 3. AWS

Folder: `aws/`

Learn:
- IAM
- EC2
- VPC
- S3
- Load Balancers

---

## 🟡 4. CI/CD

Folder: `cicd/`

Learn:
- Pipelines concept
- GitHub Actions
- GitLab CI
- Jenkins basics

---

## 🔵 5. Terraform

Folder: `terraform/`

Learn:
- Infrastructure as Code
- Providers
- State management
- Modules

---

## 🔴 6. Kubernetes

Folder: `kubernetes/`

Learn:
- Pods, Deployments, Services
- Ingress
- Scaling
- Helm
- Production patterns

---

## 🏁 Final Goal

Be able to:
- Deploy infrastructure with Terraform
- Run containerized apps
- Build CI/CD pipelines
- Manage Kubernetes clusters
- Use AWS in real-world scenarios
